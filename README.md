# TECHRADAR

## Скрипты
В корневой директории проекта можно запустить следующие скрипты:

### `npm start`
Запускает приложение в режиме разработки по адресу [http://localhost:3000](http://localhost:3000).

### `npm run build`
Запускает сборку приложения в директорию `dist`.\
В качестве сборщика используется [ViteJS](https://vitejs.dev/).

## Структура данных
В файле `src/data/data.json` содержатся данные в формате `JSON` для конфигурации и наполнения радара.

### Общее

JSON-объект имеет следующие свойства:\
`width` - ширина радара\
`height` - высота радара\
`title` - название радара\
`date` - дата последней актуализации данных\
`description` - описание радара

### Сектора

`segments` - массив секторов радара (на данный момент их должно быть строго 4)

Каждый элемент массива должен быть представлен в виде объекта со следующими полями:\
`segment.name` - название сектора\
`segment.description` - описание сектора

### Круги

`rings` - массив кругов радара (на данный момент их должно быть строго 4)

Каждый элемент массива должен быть представлен в виде объекта со следующими полями:\
`ring.name` - название круга\
`ring.color` - цветовое обозначение круга\
`ring.description` - описание круга

### Точки

`entries` - массив точек на радаре

Каждый элемент массива должен быть представлен в виде объекта со следующими полями:\
`entry.name` - название точки\
`entry.description` - описание точки\
`entry.segment` - номер сегмента, которому принадлежит точка (нумерация начинается с 0 и продолжается слева-направо, сверху-вниз)\
`entry.ring` - номер круга, которому принадлежит точка (нумерация начинается с 0 изнутри-наружу)\
`entry.moved` - направление потенциального движения точки 
(`-1` - в будущем технология будет перемещаться в более внешний круг,
`0` - перемещать технологию не планируется,
`1` - в будущем технология будет перемещаться в более внутренний круг)
